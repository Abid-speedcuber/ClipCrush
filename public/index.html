How to Use ClipCrush

From download to compressing your first Discord video in under 2 minutes.

🛠 Setup (One Time)

1

Download & Extract

Download the latest release ZIP and extract it wherever you want

2

Run setup.bat

Double-click setup.bat. It starts ClipCrush and add it to your startup apps. This app basically consumes no RAM or CPU, so don't worry.

3

You're Done!

ClipCrush is now running silently in the background, ready whenever you need it.

🎬 Compressing a Video

1

Copy Your Video

Right-click any video file and click Copy, or select it and press Ctrl+C.

2

Press the Hotkey

Press Ctrl + Alt + Shift + V. Your terminal will open and the compression will start.

3

Watch it Compress

The progress window shows the resolution, framerate, and bitrate adjustments in real time. And lets you cancel the compression if you think it's too aggressive.

4

Paste Anywhere

When done, the compressed video is in your clipboard. Press Ctrl+V to paste into Discord or anywhere else.

Things to Know:

✅ If the video is already under 10 MB, ClipCrush skips compression automatically.
⚠️ Longer videos will have more quality reduction — that's expected and unavoidable at 10 MB.
🔍 The progress window shows all changes (resolution, framerate, bitrate) before compression starts.
🛑 Press Ctrl+C in the progress window to cancel compression at any time.

Run on Startup

Double-clicking the setup.bat file should already set it to run on startup, if it doesn't:
1. Right-click ClipCrush.exe → Create Shortcut
2. Press Win+R, type shell:startup, hit Enter
3. Move the shortcut into that folder

Uninstall

1. Open Task Manager (Ctrl+Shift+Esc)
2. Find ClipCrush.exe → End Task
3. Delete the ClipCrush folder
