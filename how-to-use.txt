ClipCrush — How To Use
======================

SETUP
-----
1. Extract this folder anywhere you like (Desktop, Downloads, wherever)
2. Double-click the ClipCrush.exe in that folder
3. That's it. It's running in the background now.
   You won't see anything pop up — that's normal.


COMPRESSING A VIDEO
-------------------
1. *Copy* the video you want to compress
2. Press  Ctrl + Alt + Shift + V
3. A progress window pops up showing the compression
4. When it's done, the compressed video will be in your clipboard
5. Now you can paste the video anywhere with Ctrl+V


NOTES
-----
- Target size is under 10 MB (Discord's free upload limit)
- If the video is already under 10 MB, ClipCrush skips it
- Longer videos will have more quality reduction — that's expected
- The progress window shows exactly what's being changed
  (resolution, framerate, bitrate) before it starts
- Press Ctrl+C in the progress window to cancel at any time


RUN ON STARTUP (OPTIONAL)
--------------------------
If you want ClipCrush to start automatically with Windows:
1. Right-click ClipCrush.exe → Create shortcut
2. Press Win+R, type:  shell:startup  and hit Enter
3. Move the shortcut into the folder that pops up


UNINSTALL
---------
1. Open Task Manager with Ctrl+Shift+Esc, and then "Task Manager"
2. searchClipCrush.exe → End Task
3. Delete the ClipCrush folder you downloaded the release into
